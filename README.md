# SauceDemo Login Automation #

This is a Behavior-Driven Development (BDD) automation framework built using *Java, **Selenium WebDriver, and **Cucumber* to test the login functionality of the SauceDemo website.

## Tech Stack Used: ##
- Language: Java (JDK 24)
- Framework: Cucumber BDD
- Test Runner: TestNG
- Automation Tool: Selenium WebDriver
- Build Tool: Maven

## How to Run the Tests: ##
1. Clone this repository.
2. Open it in IntelliJ IDEA.
3. Run the Soucedemo.feature file or run mvn test in the terminal.

## 📊 Test Reporting

This framework generates detailed *Cucumber HTML Reports* after execution. It provides a clear visual representation of passed/failed steps, execution time, and test statistics.
